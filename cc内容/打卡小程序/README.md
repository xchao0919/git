# 打卡小程序

一个基于微信小程序原生 + 云开发的打卡社群管理工具。

## 功能特性

- ✅ 群组创建与管理
- ✅ 多媒体打卡（文字/图片/视频/语音）
- ✅ 细粒度权限控制
- ✅ 打卡统计与日历视图
- ✅ 打卡排行榜

## 快速开始

### 1. 配置云开发环境

1. 在微信开发者工具中打开项目
2. 开通云开发功能
3. 将 `project.config.json` 中的 `appid` 和 `app.js` 中的 `env` 替换为你的配置

### 2. 创建数据库集合

在云开发控制台创建以下集合：

- `users` - 用户表
- `groups` - 群组表
- `members` - 成员表
- `checkIns` - 打卡记录表

### 3. 部署云函数

在微信开发者工具中，右键点击 `cloudfunctions` 目录下的每个云函数文件夹，选择「上传并部署」。

## 项目结构

```
├── cloudfunctions/          # 云函数
│   ├── user/                # 用户相关
│   ├── group/               # 群组相关
│   ├── checkin/             # 打卡相关
│   └── statistics/          # 统计相关
├── miniprogram/             # 小程序代码
│   ├── pages/               # 页面
│   ├── utils/               # 工具函数
│   └── images/              # 静态图片
└── project.config.json      # 项目配置
```

## 权限说明

| 角色 | 权限 |
|------|------|
| 群主 | 全部权限 |
| 管理员 | 成员管理、打卡审核、发布公告、查看统计、数据导出 |
| 成员 | 查看统计 |

## 技术栈

- 微信小程序原生开发
- 微信云开发（云函数 + 云数据库 + 云存储）
- MongoDB 风格数据库
