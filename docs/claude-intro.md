# Claude 是什么？

**Claude** 是由 [Anthropic](https://www.anthropic.com) 开发的 AI 助手。Anthropic 是一家专注于 AI 安全研究的公司，致力于构建可靠、可解释且可对齐的 AI 系统。

## 核心特点

- **安全优先**：Claude 的训练方式强调诚实、无害和有帮助，基于 Anthropic 的"Constitutional AI"方法论。
- **强大的推理能力**：擅长逻辑推理、代码编写、数学分析和复杂问题求解。
- **长上下文窗口**：支持处理长篇文档、代码库和多轮复杂对话。
- **多语言支持**：流利使用中文、英文及多种语言。

## 主要模型系列

| 模型 | 定位 |
|------|------|
| Claude Opus | 最强能力，适合复杂任务 |
| Claude Sonnet | 能力与速度的平衡 |
| Claude Haiku | 轻量快速，适合高频调用 |

## 典型使用场景

1. **写作与编辑**：文案创作、润色、摘要
2. **代码开发**：生成、审查、调试代码
3. **知识问答**：研究、分析、解释概念
4. **数据处理**：结构化提取、格式转换
5. **对话助手**：客服、教育、个人助理

## 与 ChatGPT 的区别

Claude 更强调**安全性与诚实性**，倾向于拒绝有害请求而非迎合用户，在长文档处理和指令遵循方面表现突出。
